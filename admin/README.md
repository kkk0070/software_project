# EcoRide Admin Dashboard

A comprehensive web-based admin dashboard for the EcoRide sustainable mobility platform. Built with React, Vite, and Tailwind CSS.

## Features

The dashboard includes 14 comprehensive pages:

1. **Admin Login Page** - Secure authentication with role-based access
2. **Overview Dashboard** - Active users, rides, carbon savings, and platform health metrics
3. **User & Driver Management** - User verification, role management, and account suspension
4. **Live Ride Monitoring** - Real-time ride map, fleet distribution, and incident tracking
5. **Demand & Heatmap Analytics** - Area-wise demand, peak detection, and underserved areas
6. **Route & Performance Analytics** - Route efficiency, predicted vs actual time, fuel savings
7. **GPS & Location Accuracy** - GPS anomaly logs, accuracy trends, problematic regions
8. **Sustainability Analytics** - Emission reduction, EV impact, and community metrics
9. **AI & Optimization Control** - Demand prediction, clustering performance, model evaluation
10. **Safety & Emergency Monitoring** - SOS dashboard, live emergency locations, resolution tracking
11. **Notifications & Campaign Management** - Push notifications, campaigns, and alert scheduling
12. **Reports & Export** - Operational and sustainability reports with CSV/PDF export
13. **System Monitoring & Logs** - API health, error logs, security logs, performance metrics
14. **Settings & Configuration** - Pricing rules, feature toggles, localization, accessibility

## Getting Started

### Installation

```bash
cd admin
npm install
```

### Development

```bash
npm run dev
```

The dashboard will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

## Default Login

For testing, any email and password will work.
